## Building and Running

Git clone the project into Visual Studio IDE and 
run the code from there. 


Or from the command line:

```
$ cd ftpclient/FtpClientApp/FtpClientApp
```

 compile the following files:

```
$ csc *.cs
```

and then run it:

```
$ mono Program.exe
```
